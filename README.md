**End-to-End Retail Analytics Project**  
Uncovering purchasing patterns, customer segments, product performance, and actionable business insights from real-world shopping data using **SQL (PostgreSQL)**, **Python (Pandas)**, and **Power BI**.

## 📌 Project Overview

This portfolio project simulates a real-world retail analytics workflow. Using a dataset of customer transactions, I explored key questions such as:

- What are the top-performing product categories and items?
- How do customer demographics (age, gender, location) influence purchase behavior?
- What seasonal/time-based trends drive sales revenue?
- How can we segment customers for targeted marketing?
- Which factors most impact average order value and repeat purchases?

The full pipeline includes **data ingestion → cleaning → exploratory analysis → advanced querying → visualization → business recommendations**.

## 🛠️ Tech Stack

- **SQL (PostgreSQL)**: Data exploration, aggregations, cohort analysis, KPI calculations
- **Python (Pandas, Matplotlib/Seaborn)**: Data cleaning, EDA, feature engineering
- **Power BI**: Interactive dashboards with DAX measures, slicers, drill-downs, and KPIs
- **Tools**: Jupyter Notebook, PostgreSQL (or compatible DB), Git

## 📊 Dataset

- **File**: `customer_behavior.csv`
- **Rows**: ~3,900+ transactions (typical retail sample size)
- **Key Columns**: Customer ID, Age, Gender, Item Purchased, Category, Purchase Amount, Location, Season, Review Rating, Payment Method, etc.
- **Source**: Synthetic/anonymized retail shopping data (commonly used in analytics portfolios)

## 🔍 Key Analyses & Insights

1. **SQL Queries** (`customer_behavior_sql_queries.sql`)
   - Revenue by category, season, and location
   - Top customers by lifetime value
   - Average purchase amount by age group/gender
   - Seasonal trends and peak shopping periods

2. **Python EDA** (`Customer_Shopping_Behavior_Analysis.ipynb`)
   - Data cleaning & handling missing values
   - Distribution analysis (age, purchase amount, ratings)
   - Correlation heatmaps & visualizations
   - Customer segmentation basics (e.g., high vs low spenders)

3. **Power BI Dashboard** (`customer_behavior_dashboard.pbix`)
   - Overview KPIs: Total Revenue, Avg Order Value, Customer Count
   - Category & product performance treemaps/bar charts
   - Demographic breakdowns (age/gender slicers)
   - Seasonal & geographic sales maps/trends
   - Interactive filters for deep dives

4. **Business Deliverables**
   - `Business Problem.pdf` – Problem statement & objectives
   - `Customer Shopping Behavior Analysis.pdf` – Detailed insights & findings
   - `Customer-Shopping-Behavior-Analysis.pptx` – Presentation slides for stakeholders
